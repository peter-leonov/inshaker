Diageo Reserve World Class 2012

Осенью 2011 года в Москве стартует глобальная программа всемирно известного международного конкурса барменского мастерства Diageo Reserve World Class Russia.

Diageo Reserve World Class – международный конкурс барменского искусства, признанный во всем мире одним из самых престижных и авторитетных состязаний по созданию коктейльных напитков. Вот уже три года подряд конкурс проводится в 34 странах мира. Теперь в их число вошла и Россия. Diageo Reserve World Class – это культура барменского мастерства, способная раскрыть всю палитру красоты искусства миксологии и приготовления коктейлей.

На мировом финале Diageo Reserve World Class 2011, прошедшем в Нью-Дели 14 июля 2011 года, победителем стал Манабу Охтаке, Tower’s Bar Bellovisto (Япония).

Победа в конкурсе Diageo Reserve World Class – результат достижения высочайшего профессионального развития и первоклассного мастерства бармена. Это признание и уважение специалистов со всего мира, это шанс получить престижную награду – возможность издания собственной книги коктейлей с мировой географией распространения.

Программа Diageo Reserve World Class Russia 2012 года состоит из четырех конкурсных тематических сессий, задача которых вдохновить талантливых барменов на создание уникальных рецептур, незабываемых вкусов, на оригинальную подачу своих произведений, в основе которых алкогольные напитки класса де люкс из портфеля Diageo Reserve. В рамках каждой сессии будут проведены мастер-классы всемирно известных коктейльных гуру и бренд-амбассадоров Diageo, а также конкурсные этапы между барменами-участниками Diageo Reserve World Class 2012, согласно следующим темам конкурса: Classics with a Twist, Gentelmen’s Drinks, Asian Inspired Flavours, Drinks Inspired by Great Ladies.

География проведения мастер-классов и конкурсных этапов – Москва и Санкт-Петербург.

После окончания всех конкурсных этапов завершать глобальную программу Diageo Reserve World Class Russia 2012 будет мастер-класс бренд-амбассадора конкурса Diageo Reserve World Class Спайка Мерчанта.

Конкурс Diageo Reserve World Class Russia имеет общенациональный статус и принимает к участию барменов со всей России.

Победитель российской программы конкурса будет представлять нашу страну на международном финале Diageo Reserve World Class 2012 в числе финалистов региональных этапов со всего мира.

Регламент участия и механика приема заявок опубликованы на странице Diageo Reserve World Class Russia на Facebook и на официальном сайте Конкурса.
