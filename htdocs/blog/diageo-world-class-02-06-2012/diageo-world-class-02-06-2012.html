Diageo Reserve World Class 2012

  После виртуальной оценки авторских рецептур барменов на тему Четвертой конкурсной сессии «Великие женщины – великие напитки», определен список участников финального конкурсного дня российской программы этого года!

Ими стали:

1. Кирилл Рунков – «Сохо Лаунж», Санкт-Петербург

2. Владислав Дубинин – «Шуры-Муры», Волгоград

4. Денис Кряжев – «Сиксти», Москва

14. Иван Владимиров – «Нью Бар», Екатеринбург

15. Владимир Колганов – «Мартинез бар», Москва

Так кто же из 160 российских барменов-участников будет признан лучшим и представит Россию на глобальном Финале Diageo Reserve World Class 2012 в Рио-де-Жанейро?  Ответ узнаем совсем скоро!

Четвертый конкурсный этап Diageo Reserve World Class Russia 2012 состоится 18 июня в ресторане «Манон» (ул. 1905 года, д.2).  Сбор гостей в 17.00. Вход на мероприятие свободный.

Все обновления и новости конкурсной программы регулярно публикуются на официальном сайте конкурса и на странице Diageo Reserve World Class Russia на Facebook.
