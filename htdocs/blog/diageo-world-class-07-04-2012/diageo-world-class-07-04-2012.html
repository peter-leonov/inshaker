Diageo Reserve World Class 2012

После серии мастер-классов «Лучшего Бармена Мира» по версии Diageo Reserve World Class 2011 Манабу Отаке в Москве и Санкт-Петербурге, российские бармены представили свои версии напитков, раскрывающих тематику Третьего конкурсного этапа – «Вдохновляющие ароматы Азии».

Чарующие вкусы Индии, Китая, Японии, Кореи – это целый мир экзотических ароматов, который открывает множество новых возможностей для современной миксологии. Цель Третьей тематической сессии конкурса – вдохновить российских барменов магией новых вкусов, расширить диапазон ингредиентов для создания новых коктейльных шедевров или придания пикантности традиционной классике.

По итогам виртуального судейства определены участники Третьего конкурсного этапа, чьи авторские рецептуры на тему международного коктейльного тренда «Вдохновляющие ароматы Азии» получили лучшие оценки от Манабу Отаке:
