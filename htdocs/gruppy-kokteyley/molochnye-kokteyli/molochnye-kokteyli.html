Молочные коктейли

  1. Один из самых известных молочных коктейлей в мировой культуре - «Мартин Льюис» за пять долларов, который заказала героиня Умы Турман в знаменитом фильме Квентина Тарантино «Криминальное чтиво». Её спутник в исполнении Джона Траволты недоумевал, как смесь молока и мороженого может столько стоить.
  2. В конце XV века моряки брали на борт коз, чтобы иметь возможность пить свежее молоко во время долгого плавания и даже готовить алкогольные молочные коктейли в «домашних» условиях — прямо на корабле!
  3. Водка калорийнее молочного коктейля почти в два с половиной раза.
  4. Широкое распространение молочный коктейль получил после 1930-х годов, когда впервые появились фреонные холодильники, и стало доступным приготовление домашних молочных коктейлей.
  5. Первый президент США Джордж Вашингтон ел мороженое в таких огромных количествах, что секретарь Белого дома должен был записывать рецепты его любимых молочных коктейлей с мороженым.
  6. За один раз корова даёт от 11 до 23 литров молока. Это значит, что живя в деревне, можно приготовить около 100 молочных коктейлей дома после одной дойки.
  7. Первый блендер был запатентован в 1922 году. Его изобретатель Стивен Поплавский начал разработку чудо-миксера в 1919 году по заказу компании «Arnold Electric» специально для приготовления молочных коктейлей, рецепты для которых также предоставила компания.
  8. Банановый молочный коктейль может служить доступным антидепрессантом, ведь бананы содержат типтофан и витамин В6, с помощью которых организм вырабатывает серотонин — гормон счастья.
  9. Часто молочные коктейли называют просто «милкшейки». Впервые это слово появилось в печати в 1885 году, в описании алкогольного молочного коктейля на основе виски, похожего на «Испанский эг-ног».
  10. Больше всего бананов едят в Уганде. И хотя все знают, что можно сделать молочные коктейли с бананом даже дома, только угандцы готовят банановое пиво крепостью 28%.

/ Группы коктейлей
  • Молочный коктейль «Ежевичный милкшейк» = Молоко (3%) + Ванильное мороженое + Ежевика
  • Молочный коктейль «Малиновый милкшейк» = Молоко (3%) + Ванильное мороженое + Малина
  • Молочный коктейль «Клубничный милкшейк» = Сироп клубничная конфета + Молоко (3%) + Ванильное мороженое + Клубника
  • Молочный коктейль «Шоколадный милкшейк» = Шоколадный сироп + Молоко (3%) + Ванильное мороженое + Шоколад черный
  • Молочный коктейль «Ананасово-яблочный милкшейк» = Кокосовый сироп + Молоко (3%) + Яблоко зеленое + Ананас + Лед дробленый
  • Молочный коктейль «Кефирно-малиновый милкшейк» = Кефир (1%) + Сгущенное молоко + Лайм + Малина + Лед дробленый
  • Молочный коктейль «Кефирно-облепиховый милкшейк» = Гранатовый сок + Кефир (1%) + Облепиха + Сахарный песок + Лед дробленый
  • Молочный коктейль «Манговый милкшейк с ряженкой» = Ряженка (2,5%) + Манго + Слива черная + Джем манго + Лед дробленый
  • Молочный коктейль «Сникерс милкшейк» = Шоколадный сироп + Карамельный сироп + Молоко (3%) + Ванильное мороженое + Арахис жареный
  • Молочный коктейль «Творожно-ореховый милкшейк» = Молоко (3%) + Творог + Мед + Грецкий орех + Лед дробленый
  • Молочный коктейль «Яичный милкшейк с ряженкой» = Манговый сок + Ряженка (2,5%) + Яйцо + Кардамон + Лед дробленый
  • Молочный коктейль «Кефирно-брусничный милкшейк» = Сироп бузины + Клюквенный морс + Кефир (1%) + Лимон + Брусника + Лед дробленый
  • Молочный коктейль «Кефирно-фруктовый милкшейк» = Кефир (1%) + Сливочное мороженое + Апельсин + Банан + Клубника + Лед дробленый
  • Молочный коктейль «Грушевый милкшейк» = Миндальный сироп + Вишневый сок + Молоко (3%) + Груша + Базилик зеленый + Ванильный сахарный песок + Лед дробленый
  • Молочный коктейль «Кефирно-вишневый милкшейк» = Кефир (1%) + Шоколадное мороженое + Лимон + Вишня + Инжирное варенье + Шоколад черный + Лед дробленый
