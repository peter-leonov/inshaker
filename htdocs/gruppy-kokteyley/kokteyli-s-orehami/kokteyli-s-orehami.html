Коктейли с орехами

  1. Арахис — самый питательный орех, от которого не полнеют.
  2. 20-25 орешков арахиса в день нормализуют функции кроветворных органов и разглаживают морщинки.
  3. Грецкий орех содержит витамины группы А, Е, В, Р, С, минеральные вещества: калий, натрий, фосфор, железо, магний, кальций, йод. Грецкий орех является прекрасным источником белка, который может заменить животный белок. В 100 г грецких орехов содержится больше калорий, чем в плитке шоколада.
  4. Кокос крупный орех, плод кокосовой пальмы. Обладает твердой, ворсистой, коричневой скорлупой, под которой скрывается тонкая коричневая кожура, белая мякоть и прозрачная жидкость, часто именуемая «кокосовым молоком». Чтобы открыть кокос, нужно проделать штопором отверстия в углублениях на одной из сторон кокоса и слить воду. После этого простучать молотком по всей ширине кокоса. Орех должен сам разломаться на две половинки.

Коктейли с орехами

/ Группы коктейлей
  • Коктейль с орехами «Сникерс милкшейк» = Шоколадный сироп + Карамельный сироп + Молоко (3%) + Ванильное мороженое + Арахис жареный
  • Коктейль с орехами «Творожно-ореховый милкшейк» = Молоко (3%) + Творог + Мед + Грецкий орех + Лед дробленый
  • Коктейль с орехами «Пина Колада в кокосе» = Белый ром + Сахарный сироп + Ананас + Кокосовый крем + Кокос + Лед дробленый
  • Коктейль с орехами «Пина Колада домашняя» = Белый ром + Ананасовый сок + Ананас + Сахарный песок + Кокос + Лед в кубиках
  • Коктейль с орехами «Королева пальм» = Джин + Кокосовый ликер + Сахарный сироп + Сироп корицы + Грейпфрутовый сок + Лайм + Кокос + Лед в кубиках
  • Коктейль с орехами «Пинк флойд» = Водка + Ангостура биттер + Малиновый сироп + Сахарный сироп + Лимон + Малина + Кедровые орехи + Лед в кубиках
  • Коктейль с орехами «Тыкван до» = Водка + Сахарный сироп + Ванильный сироп + Лимон + Тыква + Мята + Корица молотая + Кедровые орехи + Лед в кубиках
