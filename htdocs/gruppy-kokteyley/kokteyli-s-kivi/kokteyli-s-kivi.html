Коктейли с киви

  1. Ведутся споры считать киви фруктом или ягодой. Так или иначе, это кисло-сладкий плод с коричневой кожурой, сочной зеленой мякотью и мелкими черными семенами.
  2. Киви принято считать национальным фруктом Новой Зеландии.
  3. Является хорошим источником витаминов С, Е, РР. Кроме того, его кожица богата различными антиоксидантами.
  4. Один киви весит в среднем 100 г. Из него можно получить 70 г пюре.

  • Коктейль с киви «Пача Ибица» = Темный ром + Сахарный сироп + Киви + Джем вишневый + Лед дробленый
  • Коктейль с киви «С Фиделем» = Белый ром + Кофейный ликер + Апельсиновый ликер + Южный комфорт + Киви
  • Коктейль с киви «Тропический морс в кувшине» = Сироп маракуйя + Вода без газа + Киви + Лайм + Сахарный песок + Лед в кубиках
  • Коктейль с киви «Карибиан тач» = Белый ром + Ликер манго + Сахарный сироп + Грейпфрутовый сок + Киви + Ананас + Лед дробленый
  • Коктейль с киви «Киви лимонад в кувшине» = Грейпфрутовый сироп + Сироп бузины + Содовая + Киви + Лимон + Базилик зеленый + Лед в кубиках
  • Коктейль с киви «Компот из киви в кувшине» = Вода без газа + Киви + Ванильный сахарный песок + Гвоздика + Звездчатый анис + Корица в палочках + Лед в кубиках
  • Коктейль с киви «Сейдж романс» = Сахарный сироп + Яблочный сок + Спрайт + Киви + Лайм + Эстрагон + Лед в кубиках
  • Коктейль с киви «Баба Таня» = Водка ванильная + Игристое вино сухое + Белое вино сухое + Кленовый сироп + Киви + Лайм + Мята + Лед в кубиках
  • Коктейль с киви «Кокосовая сангрия в кувшине» = Малибу + Белое вино сухое + Сахарный сироп + Содовая + Апельсин + Киви + Клубника + Лед в кубиках
