Коктейли с черносливом

  1. Чернослив сушеная черная слива.
  2. Чернослив улучшает состояние кожи, содержит в 1,5 раза больше калия, чем бананы, а так же в нем много железа и витаминов.
  3. Одна ягода чернослива весит около 10 г.

Коктейли с черносливом

/ Группы коктейлей
  • Коктейль с черносливом «Бурбон на черносливе» = Бурбон + Сахарный песок + Чернослив
  • Коктейль с черносливом «Рокфеллер шутер» = Водка черносмородиновая + Шоколадный сироп + Чернослив
  • Коктейль с черносливом «Спайс энд айс» = Пряный ром + Курага + Чернослив + Лед в кубиках
  • Коктейль с черносливом «Рязанов» = Виски шотландский + Апельсиновый биттер + Кленовый сироп + Яблоко зеленое + Чернослив + Лед в кубиках
  • Коктейль с черносливом «Компот из чернослива в кувшине» = Вода без газа + Виноград зеленый + Тростниковый сахарный песок + Мускатный орех молотый + Ваниль в стручках + Чернослив + Лед в кубиках
  • Коктейль с черносливом «Путь вперед» = Темный ром + Пиво светлое + Ангостура биттер + Сахарный сироп + Чернослив + Курага + Лед в кубиках
  • Коктейль с черносливом «Московский сангари» = Водка + Черносмородиновый ликер + Крепленое вино красное + Медовый сироп + Апельсин + Лимон + Гвоздика + Чернослив + Лед дробленый + Лед в кубиках
