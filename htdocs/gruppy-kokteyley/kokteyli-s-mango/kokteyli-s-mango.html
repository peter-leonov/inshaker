Коктейли с манго

  1. Манго большой продолговатый фрукт с тонкой кожицей, желтой или оранжевой мякотью и с большой косточкой в центре.
  2. Содержит почти полный набор всех необходимых человеку витаминов (С, Е, В3, В5, В6, каротиноиды) и питательных веществ.
  3. Одно манго весит в среднем 480 г.
  4. Долька манго весит около 30 г. Из одной дольки можно получить 20 г пюре для приготовления коктейлей.

Коктейли с манго

/ Группы коктейлей
  • Коктейль с манго «Джамбо» = Сироп бузины + Сливки (33%) + Манго + Персик консервированный + Лед дробленый
  • Коктейль с манго «Манго слинг» = Имбирный сироп + Спрайт + Манго + Лимон + Лед в кубиках
  • Коктейль с манго «Манговый милкшейк с ряженкой» = Ряженка (2,5%) + Манго + Слива черная + Джем манго + Лед дробленый
  • Коктейль с манго «Витаминный лимонад в кувшине» = Сироп маракуйя + Апельсиновый сок + Содовая + Манго + Лайм + Лед в кубиках
  • Коктейль с манго «Пача Сан-Паулу» = Темный ром + Кокосовый ликер + Гренадин + Ананасовый сок + Манго + Лед дробленый
  • Коктейль с манго «Ацтек» = Золотая текила + Ликер маракуйя + Блю Курасао ликер + Миндальный сироп + Грейпфрутовый сок + Манго + Лед дробленый
  • Коктейль с манго «Гуд тайм герл» = Водка + Ликер манго + Сироп маракуйя + Ванильное мороженое + Маракуйя + Манго + Лед дробленый
  • Коктейль с манго «Манго коблер 317» = Белый ром + Банановый ликер желтый + Спрайт + Лайм + Манго + Лед дробленый + Лед в кубиках
  • Коктейль с манго «Диоскорид» = Джин + Самбука апельсин-манго + Ванильный сироп + Содовая + Яйцо + Лимон + Манго + Розмарин + Лед в кубиках
  • Коктейль с манго «Московский ласси» = Водка + Сахарный сироп + Яблочный сок + Вода без газа + Какао порошок + Йогурт (3%) + Манго + Огурец + Лед в кубиках
  • Коктейль с манго «Неземная любовь» = Белый ром + Апельсиновый ликер + Медовый сироп + Сироп корицы + Березовый сок + Лайм + Манго + Тимьян + Лед в кубиках
