Программист на JavaScript: обновление страницы всех баров

Изменения коснутся только страницы всех баров. Из двух частей:

1. Заголовок с общим количеством баров

Вместо фразы «Гид по лучшим коктейльным барам списком или на карте» появляется «N лучших коктейльных баров России списком или на карте». Ссылка на описание гида теряется. Количество баров считается где-то в яваскрипте.

2. Все бары одним списком

Добавляется еще один пункт «Россия» в список городов. То есть будет «Россия: куда пойти бла-бла-бла в компании бла-бла-бла». Этот пункт содержит все бары из всех городов.

В режиме «списком» бары выводятся так же, простым списком, как и в городе. Сортируются по алфавиту.

В режиме «на карте» бары выводятся так же, все разом, как и в городе. Масштаб выбирается так, чтобы было видно всю территорию России. Да, скопления баров будут выглядеть как одна точка — для начала сойдет.

Чешутся руки обновить бары? — Пишите!
